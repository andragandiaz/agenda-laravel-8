Introduction

Base URL: http://localhost

This documentation aims to provide all the information you need to work with our API.

As you scroll, you'll see code examples for working with the API in different programming languages in the dark area to the right (or as part of the content on mobile). You can switch the language used with the tabs at the top right (or from the nav menu at the top left on mobile).

Authenticating requests

This API is not authenticated.

Agenda Management

Add a new agenda

This endpoint allows a user to add a new agenda item to the system.

Example request:
curl --request POST \
    "http://localhost/api/agendas" \
    --header "Content-Type: application/json" \
    --header "Accept: application/json" \
    --data "{
    \"tanggal\": \"2024-08-27\",
    \"waktu\": \"09:00\",
    \"kegiatan\": \"Meeting with client\",
    \"tipe_acara\": \"Offline\",
    \"tempat\": \"Jakarta\",
    \"delegasi\": \"Andra\",
    \"drescode\": \"Business Casual\"
}"

const url = new URL(
    "http://localhost/api/agendas"
);

const headers = {
    "Content-Type": "application/json",
    "Accept": "application/json",
};

let body = {
    "tanggal": "2024-08-27",
    "waktu": "09:00",
    "kegiatan": "Meeting with client",
    "tipe_acara": "Offline",
    "tempat": "Jakarta",
    "delegasi": "Andra",
    "drescode": "Business Casual"
};

fetch(url, {
    method: "POST",
    headers,
    body: JSON.stringify(body),
}).then(response => response.json());

Example response (201):


{
    "success": true,
    "message": "Agenda berhasil ditambahkan",
    "data": {
        "id": 1,
        "tanggal": "2024-08-27",
        "waktu": "09:00",
        "kegiatan": "Meeting with client",
        "tipe_acara": "Offline",
        "tempat": "Jakarta",
        "delegasi": "Andra",
        "drescode": "Business Casual"
    }
}

Example response (422):


{
    "success": false,
    "message": "Ada kesalahan dalam validasi input",
    "data": {
        "tanggal": [
            "The tanggal field is required."
        ],
        "waktu": [
            "The waktu field is required."
        ],
        "kegiatan": [
            "The kegiatan field is required."
        ]
    }
}
Received response:
Request failed with error:


Tip: Check that you're properly connected to the network.
If you're a maintainer of ths API, verify that your API is running and you've enabled CORS.
You can check the Dev Tools console for debugging information.

    "http://localhost/api/register" \
    --header "Content-Type: application/json" \
    --header "Accept: application/json" \
    --data "{
    \"name\": \"Andra\",
    \"email\": \"andra@yahoo.com\",
    \"password\": \"rahasia123\",
    \"confirm_password\": \"rahasia123\"
}"

const url = new URL(
    "http://localhost/api/register"
);

const headers = {
    "Content-Type": "application/json",
    "Accept": "application/json",
};

let body = {
    "name": "Andra",
    "email": "andra@yahoo.com",
    "password": "rahasia123",
    "confirm_password": "rahasia123"
};

fetch(url, {
    method: "POST",
    headers,
